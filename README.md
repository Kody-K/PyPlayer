## Requirements
* Python 3 https://python.org/
* Audio Files to play (.mp3)
## Installation

'''
git clone https://github.com/Kody-K/PyPlayer
cd PyPlayer
python3 -m pip install -r requirements.txt
'''

## Usage

'''
python3 pyplayer.py
'''
