# PyPlayer
A Simple, Fast, Minimalist Audio Player


## Requirements
* Python 3 https://python.org/
* Audio Files to play (.mp3)
## Installation

'''bash
git clone https://github.com/Kody-K/PyPlayer
cd PyPlayer
python3 -m pip install -r requirements.txt
'''

## Usage

'''bash
python3 pyplayer.py
'''
